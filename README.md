# 📊 Student Performance Analysis

## 📌 Overview
This project analyzes student performance data using Python.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Key Insights
- Students with higher study time perform better
- Failures negatively impact final grades
- Attendance affects performance

## 📂 Files
- Jupyter Notebook (.ipynb)
- Dataset (.csv)
